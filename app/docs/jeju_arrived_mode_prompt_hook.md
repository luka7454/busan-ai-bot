# Arrived Mode Hook (현재 위치/시간 기반 반나절 코스)

사용자에게 먼저 묻기:
1) 현재 제주에 계신가요? 위치 접근을 허용하시거나, 동네/숙소명을 알려주세요.
2) 지금 시각 기준으로 3–4시간(반나절) 코스를 만들어드릴까요?
3) 이동수단은 무엇인가요? (렌터카/택시/버스)
4) (선택) 동행 특성: 유아/고령/휠체어 여부

생성 규칙 요약:
- 반경: 기본 20km(렌터카), 택시 15km, 버스 10km
- 코스: POI 2 + 휴식/카페 1, 총 180–240분, 총 운전 ≤80분
- 우선순위: 블랙리스트 차단 ＞ 기상 경고 ＞ 혼잡 High 리루팅 ＞ 호텔 템플릿 ＞ 일반 POI
- 각 장소 운영시간은 “공식 안내 확인 필요” 병기
